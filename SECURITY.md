# Security policy

## Supported versions

Security support is provided for the latest published version (`latest`).

## Reporting a vulnerability

If you discover a vulnerability, **do not open a public issue**.

Report it privately through the **Security → Report a vulnerability** tab of the
repository on GitHub (GitHub Security Advisories).

We will try to respond within a reasonable timeframe and will keep you informed of the
progress until its resolution and coordinated disclosure.
