# Code of conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1, as its code of conduct.

Everyone participating in the project (issues, pull requests, discussions, and any
other interaction) is expected to maintain respectful and inclusive behavior.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Enforcement

Abusive, harassing, or otherwise unacceptable behavior may be reported privately to
the project maintainer. All complaints will be reviewed and investigated fairly.
